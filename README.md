# 🌐 Web Development Journey

A personal repository documenting my learning journey through web development — from HTML basics to JavaScript and Tailwind CSS.

---

## 📁 Folder Structure

```
Web-devlopment/
├── HTML/           # HTML fundamentals and practice files
├── TailwindCSS/    # Utility-first CSS styling with Tailwind
└── javaScript-main/  # JavaScript concepts and exercises
```

---

## 🛠️ Technologies Covered

- **HTML** — Page structure, semantic elements, forms, tables
- **CSS** — Styling, layouts, flexbox, animations
- **Tailwind CSS** — Utility-first CSS framework
- **JavaScript** — DOM manipulation, events, functions, and more

---

## 🚀 Getting Started

Clone the repo and open any `.html` file directly in your browser — no build step needed for most files.

```bash
git clone https://github.com/DivyanshVashishtth/Web-devlopment.git
cd Web-devlopment
```

For Tailwind CSS files, a CDN link is used so they work out of the box too.
